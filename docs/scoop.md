# Scoop Installation (Windows)

This documentation has moved to **[docs.cli.vulnetix.com/docs/getting-started/scoop](https://docs.cli.vulnetix.com/docs/getting-started/scoop/)**.

## Quick Start

```powershell
scoop bucket add vulnetix https://github.com/Vulnetix/scoop-bucket
scoop install vulnetix
vulnetix auth login
```

## Upgrading

```powershell
scoop update vulnetix
```
