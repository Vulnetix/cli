# Build from Source

This documentation has moved to **[docs.cli.vulnetix.com/docs/getting-started/from-source](https://docs.cli.vulnetix.com/docs/getting-started/from-source/)**.

## Quick Start

```bash
git clone https://github.com/Vulnetix/cli.git
cd cli
go build -o vulnetix .
./vulnetix auth login
```

Requires Go 1.21+.
