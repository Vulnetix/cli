# Homebrew Installation

This documentation has moved to **[docs.cli.vulnetix.com/docs/getting-started/homebrew](https://docs.cli.vulnetix.com/docs/getting-started/homebrew/)**.

## Quick Start

```bash
brew tap vulnetix/tap
brew install vulnetix
vulnetix auth login
```

## Upgrading

```bash
brew upgrade vulnetix
```
