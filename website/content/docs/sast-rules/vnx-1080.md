## Overview

Detects source files exceeding 1000 lines of code. Excessively large files are difficult to review for security issues, encourage mixing of concerns, and make it harder to identify where security controls are applied or bypassed.

**Severity:** Low | **CWE:** [CWE-1080](https://cwe.mitre.org/data/definitions/1080.html)

## Why This Matters

Review the flagged patterns and assess whether they represent security risks in your codebase. Even quality/maintainability weaknesses can have security implications when they affect how security-critical code is written and audited.

## Remediation

1. Review all flagged instances in context.
2. Refactor code to follow the recommended patterns described in the rule metadata.
3. Consult [CWE-1080](https://cwe.mitre.org/data/definitions/1080.html) for authoritative guidance.
