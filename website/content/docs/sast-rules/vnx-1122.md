## Overview

VNX-1122 maps to [CWE-1122: Excessive Halstead Complexity](https://cwe.mitre.org/data/definitions/1122.html). Detects source patterns associated with CWE-1122 (Excessive Halstead Complexity). Each finding should be manually reviewed for exploitability in context.

**Severity:** Low | **CWE:** [CWE-1122](https://cwe.mitre.org/data/definitions/1122.html) | **Languages:** python

## Why This Matters

This weakness class (Excessive Halstead Complexity) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-1122 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```python
// FLAGGED: contains 'def ' pattern
def 
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-1122 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-1122` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
