## Overview

VNX-1395 maps to [CWE-1395: CWE-1395](https://cwe.mitre.org/data/definitions/1395.html). Detects source patterns associated with CWE-1395 (CWE-1395). Each finding should be manually reviewed for exploitability in context.

**Severity:** Low | **CWE:** [CWE-1395](https://cwe.mitre.org/data/definitions/1395.html) | **Languages:** multiple

## Why This Matters

This weakness class (CWE-1395) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-1395 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```
// FLAGGED: contains '"lodash": "<4.17.12"' pattern
"lodash": "<4.17.12"
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-1395 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-1395` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
