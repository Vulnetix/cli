---
title: "VNX-1204 – Generation of Weak Initialization Vector"
description: "Detect static, zero-byte, or hardcoded initialization vectors and nonces used for symmetric encryption, which break semantic security of cipher modes like CBC, CTR, and GCM."
---

## Overview

An initialization vector (IV) or nonce must be unique and unpredictable for every encryption operation. Using a hardcoded all-zero IV, a static byte string, or ECB mode (which uses no IV) causes the same plaintext to always produce the same ciphertext, enabling plaintext recovery and ciphertext forgery attacks. **Severity:** High | **CWE:** [CWE-1204 – Generation of Weak Initialization Vector](https://cwe.mitre.org/data/definitions/1204.html)

## Why This Matters

CBC mode with a predictable IV is vulnerable to the BEAST attack and chosen-plaintext attacks. CTR and GCM modes with reused nonces completely break confidentiality and integrity — reusing a GCM nonce leaks the authentication key. ECB mode produces identical ciphertext blocks for identical plaintext blocks, leaking patterns (the classic "ECB penguin" image).

## What Gets Flagged

```python
# FLAGGED: All-zero IV
from Crypto.Cipher import AES
iv = b'\x00' * 16
cipher = AES.new(key, AES.MODE_CBC, iv)

# FLAGGED: Hardcoded IV literal
nonce = b'\x01\x02\x03\x04\x05\x06\x07\x08\x09\x0a\x0b\x0c'
```

```java
// FLAGGED: IvParameterSpec with zero byte array
byte[] iv = new byte[16];  // all zeros
IvParameterSpec ivSpec = new IvParameterSpec(iv);

// FLAGGED: ECB mode
Cipher cipher = Cipher.getInstance("AES/ECB/PKCS5Padding");
```

## Remediation

```python
# SAFE: Random IV generated per encryption
import os
from Crypto.Cipher import AES

key = os.urandom(32)  # 256-bit key
iv = os.urandom(16)   # fresh random IV
cipher = AES.new(key, AES.MODE_CBC, iv)
# Prepend IV to ciphertext for decryption
ciphertext = iv + cipher.encrypt(data)

# PREFERRED: Use GCM for authenticated encryption
nonce = os.urandom(12)
cipher = AES.new(key, AES.MODE_GCM, nonce=nonce)
ciphertext, tag = cipher.encrypt_and_digest(data)
```

```java
// SAFE: Random IV using SecureRandom
byte[] iv = new byte[16];
new SecureRandom().nextBytes(iv);
IvParameterSpec ivSpec = new IvParameterSpec(iv);
Cipher cipher = Cipher.getInstance("AES/GCM/NoPadding");
cipher.init(Cipher.ENCRYPT_MODE, secretKey, new GCMParameterSpec(128, iv));
```
