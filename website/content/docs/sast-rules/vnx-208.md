---
title: "VNX-208 – Observable Timing Discrepancy (Timing Attack)"
description: "Detect direct equality comparisons of secrets, tokens, or HMACs that leak information through response time differences."
---

## Overview

This rule flags comparisons of secret values (tokens, passwords, HMACs, API keys) using standard equality operators (`==`, `===`, `.equals()`). These operators short-circuit on the first differing byte, causing measurable timing differences that reveal how many bytes of the secret an attacker guessed correctly. By making many requests and measuring response times, an attacker can recover the full secret one byte at a time. This maps to [CWE-208: Observable Timing Discrepancy](https://cwe.mitre.org/data/definitions/208.html).

**Severity:** Medium | **CWE:** [CWE-208 – Observable Timing Discrepancy](https://cwe.mitre.org/data/definitions/208.html)

## Why This Matters

Timing attacks have been demonstrated against real-world authentication systems, CSRF token validation, and API key verification in multiple languages. A typical attack requires tens of thousands of requests but is feasible from adjacent networks or via cached connections. The fix is a single function call replacement in all affected languages.

## What Gets Flagged

```python
# FLAGGED: Python — direct equality
if token == stored_token:
    return True
```

```go
// FLAGGED: Go — direct equality of tokens
if token == storedToken {
    http.Error(w, "ok", 200)
}
```

```java
// FLAGGED: Java — String.equals() for HMAC
if (digest.equals(expectedDigest)) {
    return true;
}
```

```php
<?php
// FLAGGED: PHP — == comparison for token
if ($token == $stored) {
    $authenticated = true;
}
```

```ruby
# FLAGGED: Ruby — direct equality for password
if password == stored_password
  authenticate!
end
```

```javascript
// FLAGGED: Node.js — === for API key
if (token === storedToken) {
    next();
}
```

## Remediation

Replace short-circuit equality with a constant-time comparison function:

```python
# SAFE: Python
import hmac
if hmac.compare_digest(token.encode(), stored_token.encode()):
    return True
```

```go
// SAFE: Go
import "crypto/subtle"
if subtle.ConstantTimeCompare([]byte(token), []byte(storedToken)) == 1 {
    // authenticated
}
```

```java
// SAFE: Java
import java.security.MessageDigest;
if (MessageDigest.isEqual(token.getBytes(), storedToken.getBytes())) {
    return true;
}
```

```php
<?php
// SAFE: PHP
if (hash_equals($stored, $token)) {
    $authenticated = true;
}
```

```ruby
# SAFE: Ruby
require 'active_support/security_utils'
if ActiveSupport::SecurityUtils.secure_compare(password, stored_password)
  authenticate!
end
```

```javascript
// SAFE: Node.js
const crypto = require('crypto');
const a = Buffer.from(token);
const b = Buffer.from(storedToken);
if (a.length === b.length && crypto.timingSafeEqual(a, b)) {
    next();
}
```
