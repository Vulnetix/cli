title: "VNX-257 – Storing Passwords in Recoverable Format"
description: "Detect reversible encryption or Base64 encoding used for password storage."
---

## Overview

Using AES encryption or Base64 encoding for passwords provides false security — the key or encoding scheme can be extracted from the application, recovering all passwords. Only one-way hash functions with key derivation (bcrypt, argon2, scrypt) are appropriate for credential storage. This maps to [CWE-257: Storing Passwords in a Recoverable Format](https://cwe.mitre.org/data/definitions/257.html).

**Severity:** High | **CWE:** [CWE-257](https://cwe.mitre.org/data/definitions/257.html)

## Remediation

```python
# SAFE: argon2 one-way hash
from argon2 import PasswordHasher
ph = PasswordHasher(time_cost=2, memory_cost=65536, parallelism=2)
stored = ph.hash(password)
```
