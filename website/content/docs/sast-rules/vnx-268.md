## Overview

VNX-268 maps to [CWE-268: Privilege Chaining](https://cwe.mitre.org/data/definitions/268.html). Detects source patterns associated with CWE-268 (Privilege Chaining). Each finding should be manually reviewed for exploitability in context.

**Severity:** Low | **CWE:** [CWE-268](https://cwe.mitre.org/data/definitions/268.html) | **Languages:** c, cpp, python

## Why This Matters

This weakness class (Privilege Chaining) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-268 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```python
// FLAGGED: contains 'sudo' pattern
sudo
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-268 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-268` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
