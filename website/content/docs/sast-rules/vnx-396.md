## Why This Matters

This weakness class (Declaration of Catch for Generic Exception) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-396 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.
