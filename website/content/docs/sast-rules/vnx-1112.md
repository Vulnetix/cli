## Overview

VNX-1112 maps to [CWE-1112: Incomplete Documentation of Program Execution](https://cwe.mitre.org/data/definitions/1112.html). Detects source patterns associated with CWE-1112 (Incomplete Documentation of Program Execution). Each finding should be manually reviewed for exploitability in context.

**Severity:** Low | **CWE:** [CWE-1112](https://cwe.mitre.org/data/definitions/1112.html) | **Languages:** python

## Why This Matters

This weakness class (Incomplete Documentation of Program Execution) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-1112 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-1112 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-1112` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
