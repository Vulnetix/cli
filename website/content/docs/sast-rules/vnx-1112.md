## Overview

Detects known vulnerability references (CVE, CWE, GHSA) in comments without corresponding remediation notes or tracking. Acknowledged but unresolved vulnerabilities documented only in comments may never be fixed.

**Severity:** Low | **CWE:** [CWE-1112](https://cwe.mitre.org/data/definitions/1112.html)

## Why This Matters

Review the flagged patterns in context. Even quality and maintainability weaknesses can have security implications when they affect how security-critical code is structured, audited, and tested.

## Remediation

1. Review all flagged instances in context.
2. Refactor code to follow the recommended patterns described in the rule.
3. Consult [CWE-1112](https://cwe.mitre.org/data/definitions/1112.html) for authoritative guidance.
