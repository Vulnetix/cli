---
title: "VNX-767 – Access to Critical Private Variable via Public Method"
description: "Detect Java public getters that return mutable internal collections of security-sensitive data, allowing callers to modify internal state."
---

## Overview

This rule flags Java public methods that return mutable collections (`List`, `Map`, `Set`) or arrays (`char[]`) of security-sensitive data. Returning a reference to the internal collection allows callers to modify it directly, bypassing validation and access controls. This maps to [CWE-767: Access to Critical Private Variable via Public Method](https://cwe.mitre.org/data/definitions/767.html).

**Severity:** Low | **CWE:** [CWE-767](https://cwe.mitre.org/data/definitions/767.html)

## Why This Matters

Returning the internal collection reference means external code can add, remove, or modify elements without going through any validation logic. This is a form of broken encapsulation that leads to inconsistent state, bypassed access controls, and unexpected privilege escalation.

## What Gets Flagged

```java
// FLAGGED: returns mutable internal list
public List<Credential> getCredentials() {
    return this.credentials;  // caller can add/remove/modify!
}

// FLAGGED: returns internal char[] password
public char[] getPassword() {
    return this.password;  // caller holds a reference to internal array
}
```

## Remediation

```java
// SAFE: return an unmodifiable view
public List<Credential> getCredentials() {
    return Collections.unmodifiableList(this.credentials);
}

// SAFE: return a defensive copy
public List<Credential> getCredentials() {
    return new ArrayList<>(this.credentials);
}

// SAFE: for char[], return a copy and document to zero after use
public char[] getPassword() {
    return Arrays.copyOf(this.password, this.password.length);
    // Caller is responsible for zeroing: Arrays.fill(copy, '\0');
}

// BEST: don't expose credentials at all — use a verify method instead
public boolean verifyPassword(char[] submitted) {
    return Arrays.equals(this.password, submitted);
}
```
