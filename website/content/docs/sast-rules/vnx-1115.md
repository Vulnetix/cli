## Overview

Detects Windows-style CRLF line endings in shell scripts and other files where CR/LF can cause execution failures or unexpected behaviour. CRLF in shell scripts causes '\\r' to be appended to commands, breaking execution and potentially skipping security checks.

**Severity:** Low | **CWE:** [CWE-1115](https://cwe.mitre.org/data/definitions/1115.html)

## Why This Matters

Review the flagged patterns in context. Even quality and maintainability weaknesses can have security implications when they affect how security-critical code is structured, audited, and tested.

## Remediation

1. Review all flagged instances in context.
2. Refactor code to follow the recommended patterns described in the rule.
3. Consult [CWE-1115](https://cwe.mitre.org/data/definitions/1115.html) for authoritative guidance.
