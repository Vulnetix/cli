## Overview

Detects generated code files that rely on specific runtime library versions without explicit version pinning. Generated code that assumes a specific runtime component version can break or introduce vulnerabilities when the runtime is updated independently.

**Severity:** Low | **CWE:** [CWE-1101](https://cwe.mitre.org/data/definitions/1101.html)

## Why This Matters

Review the flagged patterns in context. Even quality and maintainability weaknesses can have security implications when they affect how security-critical code is structured, audited, and tested.

## Remediation

1. Review all flagged instances in context.
2. Refactor code to follow the recommended patterns described in the rule.
3. Consult [CWE-1101](https://cwe.mitre.org/data/definitions/1101.html) for authoritative guidance.
