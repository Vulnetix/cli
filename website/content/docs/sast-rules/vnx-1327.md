---
title: "VNX-1327 – Binding to an Unrestricted IP Address"
description: "Detect server sockets bound to 0.0.0.0 or all-interfaces wildcard addresses, which expose services on every network interface including public-facing ones."
---

## Overview

Binding a server socket to `0.0.0.0` (IPv4 wildcard) or `::` (IPv6 wildcard) accepts connections on all network interfaces — including public-facing ones. Services intended only for internal use or localhost become accessible to any host that can reach any interface on the machine. This is particularly dangerous in cloud environments where virtual machines have both internal and public interfaces. **Severity:** Medium | **CWE:** [CWE-1327 – Binding to an Unrestricted IP Address](https://cwe.mitre.org/data/definitions/1327.html)

## Why This Matters

Redis, MongoDB, and Memcached have all had major breaches caused by services bound to `0.0.0.0` in cloud environments. The 2017 MongoDB ransomware attacks affected approximately 30,000 servers exposed this way. Development servers accidentally deployed to production with `host='0.0.0.0'` become internet-accessible. Go's `net.Listen("tcp", ":8080")` silently binds all interfaces.

## What Gets Flagged

```python
# FLAGGED: Flask bound to all interfaces
app.run(host='0.0.0.0', port=5000)

# FLAGGED: Python socket binding
s.bind(('0.0.0.0', 8080))
```

```javascript
// FLAGGED: Node.js HTTP server
server.listen(3000, '0.0.0.0');
```

```go
// FLAGGED: Go net.Listen binds all interfaces
listener, err := net.Listen("tcp", ":8080")
```

## Remediation

```python
# SAFE: Bind to localhost only for internal services
app.run(host='127.0.0.1', port=5000)

# Or use environment variable to allow flexible deployment
import os
host = os.environ.get('BIND_HOST', '127.0.0.1')
app.run(host=host, port=5000)
```

```javascript
// SAFE: Bind to localhost
server.listen(3000, '127.0.0.1', () => {
    console.log('Server listening on localhost:3000');
});
```

```go
// SAFE: Explicit localhost bind
listener, err := net.Listen("tcp", "127.0.0.1:8080")

// Or from environment
addr := os.Getenv("BIND_ADDR")
if addr == "" {
    addr = "127.0.0.1:8080"
}
listener, err := net.Listen("tcp", addr)
```
