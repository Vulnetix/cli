## Overview

VNX-1327 maps to [CWE-1327: Binding to an Unrestricted IP Address](https://cwe.mitre.org/data/definitions/1327.html). Detects source patterns associated with CWE-1327 (Binding to an Unrestricted IP Address). Each finding should be manually reviewed for exploitability in context.

**Severity:** Medium | **CWE:** [CWE-1327](https://cwe.mitre.org/data/definitions/1327.html) | **Languages:** go, node, python

## Why This Matters

This weakness class (Binding to an Unrestricted IP Address) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-1327 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```python
// FLAGGED: contains '0.0.0.0' pattern
0.0.0.0
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-1327 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-1327` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
