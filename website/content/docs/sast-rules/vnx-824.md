---
title: "VNX-824 – Access of Uninitialized Pointer"
description: "Detect C pointer declarations without initialization that are subsequently dereferenced, leading to undefined behavior."
---

## Overview

This rule flags C pointer variables declared without initialization (e.g., `int *ptr;` rather than `int *ptr = NULL;`) that are then immediately dereferenced or passed to a function on the next line. Dereferencing an uninitialized pointer reads whatever garbage value is on the stack at that location, which may be an attacker-controlled value or a crash-inducing address. This maps to [CWE-824: Access of Uninitialized Pointer](https://cwe.mitre.org/data/definitions/824.html).

**Severity:** High | **CWE:** [CWE-824](https://cwe.mitre.org/data/definitions/824.html)

## Why This Matters

On the stack, uninitialized pointers contain leftover values from previous function calls. In certain attack scenarios (heap spraying, stack manipulation), an attacker can influence what value is there and cause the program to execute code or access controlled memory locations when the pointer is dereferenced. Even without an attacker, accidental use of uninitialized pointers causes non-deterministic crashes that are hard to reproduce.

## What Gets Flagged

```c
// FLAGGED: pointer declared without initialization, then dereferenced
int *ptr;
*ptr = 42;  // ptr is uninitialized

// FLAGGED: char pointer then used in strcpy
char *buf;
strcpy(buf, user_input);  // buf points to garbage

// FLAGGED: struct pointer used without allocation
struct Node *node;
node->value = 10;  // node is uninitialized
```

## Remediation

```c
// SAFE: initialize pointer to NULL
int *ptr = NULL;
// Then check before use:
if (ptr != NULL) {
    *ptr = 42;
}

// SAFE: allocate before use
char *buf = malloc(BUFFER_SIZE);
if (buf == NULL) {
    return -1;
}
strncpy(buf, user_input, BUFFER_SIZE - 1);
buf[BUFFER_SIZE - 1] = '\0';
free(buf);

// SAFE: allocate struct before use
struct Node *node = malloc(sizeof(struct Node));
if (node == NULL) {
    return -1;
}
memset(node, 0, sizeof(*node));  // zero-initialize
node->value = 10;
```

Initialize all pointer variables at declaration time. Enable compiler warnings for uninitialized variable usage (`-Wuninitialized` in GCC/Clang). Use static analysis tools such as Coverity or PVS-Studio to catch these at build time.
