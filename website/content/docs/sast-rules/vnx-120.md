---

## Overview

CWE-120 is the canonical description of the classic stack and heap buffer overflow. It covers every situation where a copy operation writes data into a fixed-size buffer without first verifying that the source fits. The most common offenders — `strcpy`, `strcat`, `gets`, and `memcpy` with a size derived from the source rather than the destination — are flagged by this rule. An attacker who can supply input longer than the buffer can overwrite adjacent memory, corrupt control flow data, and in many cases achieve arbitrary code execution.

**Severity:** Critical | **CWE:** [CWE-120 – Buffer Copy without Checking Size of Input ('Classic Buffer Overflow')](https://cwe.mitre.org/data/definitions/120.html)

## Why This Matters

CWE-120 has ranked in the MITRE Top 25 Most Dangerous Software Weaknesses every year since the list was introduced. It underpins decades of high-severity CVEs across operating systems, network daemons, embedded firmware, and game engines. The consequence is almost always a crash (DoS) at minimum, and frequently remote code execution when combined with a memory-disclosure primitive. Stack canaries, ASLR, and NX pages mitigate exploitation but cannot prevent it on all platforms. In safety-critical and embedded systems, these mitigations are often absent entirely.

## What Gets Flagged

```c
/* FLAGGED: strcpy copies until null byte — no destination size check */
char dest[32];
strcpy(dest, user_supplied_string);

/* FLAGGED: strcat appends until null — accumulation easily overflows */
char greeting[64] = "Hello ";
strcat(greeting, username);

/* FLAGGED: gets reads without any length limit — banned in C11 */
char buf[128];
gets(buf);

/* FLAGGED: memcpy size taken from source, not checked against dest */
char dst[64];
memcpy(dst, src, strlen(src));  // dst may be smaller than src
```

## Remediation

```c
#include <string.h>
#include <stdio.h>

/* SAFE: strlcpy — copies at most dstsize-1 bytes, always terminates */
char dest[32];
strlcpy(dest, user_supplied_string, sizeof(dest));

/* SAFE: strlcat — appends respecting remaining capacity */
char greeting[64] = "Hello ";
strlcat(greeting, username, sizeof(greeting));

/* SAFE: fgets — bounded read from any FILE*, strip newline */
char buf[128];
fgets(buf, sizeof(buf), stdin);
buf[strcspn(buf, "\n")] = '\0';

/* SAFE: memcpy with explicit destination size */
char dst[64];
size_t to_copy = strlen(src);
if (to_copy >= sizeof(dst)) { to_copy = sizeof(dst) - 1; }
memcpy(dst, src, to_copy);
dst[to_copy] = '\0';
```

Enable AddressSanitizer (`-fsanitize=address`) and compiler hardening (`-D_FORTIFY_SOURCE=2`) to detect overflows in tests.
