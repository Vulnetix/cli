## Overview

VNX-419 maps to [CWE-419: Unprotected Primary Channel](https://cwe.mitre.org/data/definitions/419.html). Detects source patterns associated with CWE-419 (Unprotected Primary Channel). Each finding should be manually reviewed for exploitability in context.

**Severity:** Medium | **CWE:** [CWE-419](https://cwe.mitre.org/data/definitions/419.html) | **Languages:** go, java

## Why This Matters

This weakness class (Unprotected Primary Channel) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-419 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```java
// FLAGGED: contains 'ServerSocket(' pattern
ServerSocket(
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-419 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-419` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
