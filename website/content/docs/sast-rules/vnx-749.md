---
title: "VNX-749 – Exposed Dangerous Method or Function"
description: "Detect public methods that perform destructive or privileged operations without access control checks."
---

## Overview

This rule flags public methods and API routes with names suggesting dangerous operations (delete all records, drop table, reset passwords, disable authentication) that lack visible access control guards. Exposing these operations without authorization checks allows any authenticated — or even unauthenticated — user to invoke them. This maps to [CWE-749: Exposed Dangerous Method or Function](https://cwe.mitre.org/data/definitions/749.html).

**Severity:** Medium | **CWE:** [CWE-749](https://cwe.mitre.org/data/definitions/749.html)

## Why This Matters

Destructive operations exposed without access control are a common source of mass data deletion incidents. Admin endpoints accidentally left accessible to regular users have been exploited in numerous high-profile breaches.

## What Gets Flagged

```java
// FLAGGED: dangerous public method without access control annotation
public void deleteAllUsers() {
    userRepository.deleteAll();
}

// FLAGGED: admin method without @PreAuthorize
public void disableSecurity() {
    securityConfig.disable();
}
```

```python
# FLAGGED: dangerous function without auth check
def delete_all():
    db.session.query(User).delete()
    db.session.commit()
```

```javascript
// FLAGGED: admin route without middleware
app.delete('/admin/deleteAllUsers', (req, res) => {
    db.users.deleteMany({});
    res.send('done');
});
```

## Remediation

```java
// SAFE: protected with role check
@PreAuthorize("hasRole('SUPER_ADMIN')")
@AuditLog
public void deleteAllUsers() {
    auditService.log("deleteAllUsers called by " + currentUser());
    userRepository.deleteAll();
}
```

```javascript
// SAFE: authentication and authorization middleware applied
app.delete('/admin/deleteAllUsers',
    requireAuth,
    requireRole('super_admin'),
    auditLog,
    async (req, res) => {
        await db.users.deleteMany({});
        res.send('done');
    }
);
```
