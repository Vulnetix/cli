## Overview

This rule flags direct string equality comparisons of role names, paths, or permissions without case normalization. A role check like `role == "admin"` is bypassed by supplying `"Admin"` or `"ADMIN"`. This maps to [CWE-178: Improper Handling of Case Sensitivity](https://cwe.mitre.org/data/definitions/178.html).

**Severity:** Medium | **CWE:** [CWE-178](https://cwe.mitre.org/data/definitions/178.html)

## Remediation

```python
# SAFE: normalize before comparing
if role.lower() == 'admin':
    grant_access()
```

```php
<?php
// SAFE: strtolower before comparison
if (strtolower($role) === 'admin') {
    grantAccess();
}
```
