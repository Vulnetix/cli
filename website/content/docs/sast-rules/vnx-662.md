---
title: "VNX-662 – Improper Synchronization"
description: "Detect shared mutable authentication state accessed from multiple threads without proper synchronization."
---

## Overview

When authentication or authorization state is stored in shared mutable variables (static fields, module-level variables, shared maps) without synchronization primitives (locks, mutexes, atomic types), concurrent requests can see inconsistent state. A thread reading `authenticated = true` while another thread is resetting it can result in unauthorized access or privilege escalation.

**Severity:** Medium | **CWE:** [CWE-662 – Improper Synchronization](https://cwe.mitre.org/data/definitions/662.html)

## Why This Matters

In Java, a `static boolean authenticated` field accessed by multiple threads without `volatile` or synchronization can be cached in CPU registers, meaning one thread's write is not visible to another. In Go, concurrent map writes without a mutex cause a runtime panic (data race). These bugs are non-deterministic and difficult to reproduce in testing but cause real security failures in production.

## What Gets Flagged

```java
// FLAGGED: Static boolean without volatile — not thread-safe
public class AuthManager {
    private static boolean authenticated = false;
    
    public static void login(String user, String pass) {
        authenticated = checkCredentials(user, pass);
    }
}
```

```go
// FLAGGED: map accessed from goroutines without sync.Mutex
var sessionStore = map[string]bool{}

func setSession(id string, auth bool) {
    sessionStore[id] = auth  // data race if called concurrently
}
```

## Remediation

Use proper synchronization primitives:

```java
// SAFE: volatile for visibility, or AtomicBoolean for atomicity
import java.util.concurrent.atomic.AtomicBoolean;

public class AuthManager {
    private static final AtomicBoolean authenticated = new AtomicBoolean(false);
    
    public static void login(String user, String pass) {
        authenticated.set(checkCredentials(user, pass));
    }
    
    public static boolean isAuthenticated() {
        return authenticated.get();
    }
}
```

```go
// SAFE: sync.RWMutex protects the map
import "sync"

var (
    mu           sync.RWMutex
    sessionStore = map[string]bool{}
)

func setSession(id string, auth bool) {
    mu.Lock()
    defer mu.Unlock()
    sessionStore[id] = auth
}
```
