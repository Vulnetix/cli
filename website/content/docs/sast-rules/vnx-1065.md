## Overview

Detects direct thread management, process spawning, and low-level resource management in code intended for application server environments (Jakarta EE, Spring, etc.). Application servers manage their own thread pools; directly creating threads bypasses container lifecycle management and can exhaust resources.

**Severity:** Low | **CWE:** [CWE-1065](https://cwe.mitre.org/data/definitions/1065.html)

## Why This Matters

Review the flagged patterns and assess whether they represent security risks in your codebase. Even quality/maintainability weaknesses can have security implications when they affect how security-critical code is written and audited.
