---
title: "VNX-174 – Path Equivalence (Extra Slashes or Self-Directory)"
description: "Detect path-based access control checks that can be bypassed using double slashes or dot-segment equivalences."
---

## Overview

This rule flags path prefix checks (startsWith, HasPrefix) and raw path usage without prior normalization. Paths like `//admin` or `/./api` are equivalent to `/admin` and `/api` on most filesystems but bypass prefix checks on the raw string. This maps to [CWE-174: Improper Handling of Alternate Encoding](https://cwe.mitre.org/data/definitions/174.html).

**Severity:** Medium | **CWE:** [CWE-174](https://cwe.mitre.org/data/definitions/174.html)

## Remediation

Normalize the path before any access control check:

```go
// SAFE: Go
import "path/filepath"
clean := filepath.Clean(req.URL.Path)
if !strings.HasPrefix(clean, "/admin") { ... }
```

```python
# SAFE: Python
import os
clean = os.path.normpath(request.path)
if not clean.startswith('/admin'): ...
```
