## Overview

Detects files with multiple goto statements. Excessive goto usage creates spaghetti control flow that can bypass security checks, skip cleanup code, and make security auditing extremely difficult.

**Severity:** Low | **CWE:** [CWE-1119](https://cwe.mitre.org/data/definitions/1119.html)

## Why This Matters

Review the flagged patterns in context. Even quality and maintainability weaknesses can have security implications when they affect how security-critical code is structured, audited, and tested.

## Remediation

1. Review all flagged instances in context.
2. Refactor code to follow the recommended patterns described in the rule.
3. Consult [CWE-1119](https://cwe.mitre.org/data/definitions/1119.html) for authoritative guidance.
