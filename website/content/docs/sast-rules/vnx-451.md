## Overview

VNX-451 maps to [CWE-451: CWE-451](https://cwe.mitre.org/data/definitions/451.html). Detects source patterns associated with CWE-451 (CWE-451). Each finding should be manually reviewed for exploitability in context.

## Why This Matters

This weakness class (CWE-451) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-451 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```
// FLAGGED: contains 'onerror=' pattern
onerror=
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-451 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-451` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
