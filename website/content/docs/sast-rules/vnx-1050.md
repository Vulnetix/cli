## Overview

Detects resource-intensive operations performed inside loops. Database queries, HTTP requests, file I/O, and cryptographic operations inside loops cause O(n) resource consumption that can exhaust server capacity and enable denial-of-service conditions.

**Severity:** Low | **CWE:** [CWE-1050 – Excessive Platform Resource Consumption within a Loop](https://cwe.mitre.org/data/definitions/1050.html)

## Why This Matters

A loop iterating over 1,000 items that makes a database query each iteration causes 1,000 queries per request. An attacker who can control the iteration count (e.g., by providing a large list) can exhaust database connection pools, causing denial of service for all users.

## What Gets Flagged

```python
# FLAGGED: database query inside loop
for user_id in user_ids:
    user = db.query(User).filter_by(id=user_id).first()
    process(user)
```

## Remediation

```python
# SAFE: batch query outside loop
users = db.query(User).filter(User.id.in_(user_ids)).all()
for user in users:
    process(user)
```
