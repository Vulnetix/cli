## Overview

Detects non-thread-safe patterns in code used in multi-threaded contexts. This includes Python global variable mutation without locks and Go's `math/rand` global source without per-goroutine instances.

**Severity:** Low | **CWE:** [CWE-1058 – Invokable Control Element in Multi-Thread Context with non-MT-safe Implementation](https://cwe.mitre.org/data/definitions/1058.html)

## Why This Matters

Race conditions in security code can cause authentication state to leak between concurrent requests, sessions to be mixed, or random number generation to produce predictable sequences.

## What Gets Flagged

```go
// FLAGGED: math/rand global source in concurrent code
import "math/rand"
token := rand.Int63()
```

```python
# FLAGGED: unsynchronized global security state
global authenticated
authenticated = True
```

## Remediation

```go
// SAFE: per-goroutine source
src := rand.NewSource(time.Now().UnixNano())
rng := rand.New(src)
token := rng.Int63()

// Or use crypto/rand for security-sensitive values
import "crypto/rand"
```
