## Overview

VNX-368 maps to [CWE-368: Context Switching Race Condition](https://cwe.mitre.org/data/definitions/368.html). Detects source patterns associated with CWE-368 (Context Switching Race Condition). Each finding should be manually reviewed for exploitability in context.

**Severity:** Low | **CWE:** [CWE-368](https://cwe.mitre.org/data/definitions/368.html) | **Languages:** c

## Why This Matters

This weakness class (Context Switching Race Condition) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-368 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```c
// FLAGGED: contains 'semaphore' pattern
semaphore
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-368 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-368` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
