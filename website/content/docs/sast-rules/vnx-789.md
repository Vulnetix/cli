---
title: "VNX-789 – Memory Allocation with Excessive Size Value"
description: "Detect malloc/new/bytearray/Buffer.alloc calls using user-controlled size values without an upper bound check, enabling memory exhaustion DoS."
---

## Overview

This rule flags memory allocations where the size argument derives from user input (via `atoi()`, `Integer.parseInt()`, `int(request...)`) without a preceding maximum size check. An attacker can request allocations of terabytes of memory, exhausting RAM and swap and causing the process to be killed. This maps to [CWE-789: Memory Allocation with Excessive Size Value](https://cwe.mitre.org/data/definitions/789.html).

**Severity:** High | **CWE:** [CWE-789](https://cwe.mitre.org/data/definitions/789.html)

## Why This Matters

An allocation of `malloc(atoi(user_input))` where `user_input` is "2147483647" (INT_MAX) will attempt to allocate 2GB. Sending this request in a tight loop from multiple clients causes rapid OOM. Additionally, integer overflow in size calculations (e.g., `n * sizeof(struct)`) can produce under-allocations followed by heap overflow writes.

## What Gets Flagged

```c
// FLAGGED: malloc with user-controlled size
void handle_request(char *size_str) {
    int n = atoi(size_str);
    char *buf = malloc(n);  // n could be GB or trigger integer overflow
}
```

```java
// FLAGGED: array allocation with user input
int size = Integer.parseInt(request.getParameter("size"));
byte[] buf = new byte[size];  // OOM if size is large
```

```python
# FLAGGED: bytearray with user input
size = int(request.GET['size'])
data = bytearray(size)  // OOM if size is huge
```

```javascript
// FLAGGED: Buffer with user input
const size = parseInt(req.query.size);
const buf = Buffer.alloc(size);  // DoS if size is unreasonable
```

## Remediation

```c
// SAFE: validate size before allocating
#define MAX_ALLOC_SIZE (1024 * 1024)  // 1MB maximum
int n = atoi(size_str);
if (n <= 0 || n > MAX_ALLOC_SIZE) {
    send_error(client, "Invalid size");
    return;
}
char *buf = malloc(n);
if (buf == NULL) { /* handle OOM */ }
```

```java
// SAFE: check bounds before allocation
int size = Integer.parseInt(request.getParameter("size"));
if (size < 1 || size > MAX_ALLOWED_SIZE) {
    throw new IllegalArgumentException("Invalid size: " + size);
}
byte[] buf = new byte[size];
```

```python
# SAFE: cap the size
MAX_SIZE = 1024 * 1024  # 1MB
size = min(max(0, int(request.GET['size'])), MAX_SIZE)
data = bytearray(size)
```
