description: "Detect use of fast cryptographic hash functions (MD5, SHA-1, SHA-256) for password storage instead of purpose-built slow hashing algorithms like bcrypt, Argon2, or PBKDF2."
---

## Overview

This rule flags code that uses general-purpose hash functions — MD5, SHA-1, SHA-256 — to hash passwords for storage. These algorithms are designed to be fast and are not appropriate for password storage. A modern GPU can compute billions of SHA-256 hashes per second, making offline brute-force attacks against a leaked database trivially fast. Purpose-built password hashing algorithms (bcrypt, Argon2, PBKDF2, scrypt) are deliberately slow and memory-intensive to resist this class of attack. This maps to [CWE-916: Use of Password Hash With Insufficient Computational Effort](https://cwe.mitre.org/data/definitions/916.html).

**Severity:** High | **CWE:** [CWE-916 – Use of Password Hash With Insufficient Computational Effort](https://cwe.mitre.org/data/definitions/916.html)

## Why This Matters

When a database of hashed passwords is leaked (a frequent occurrence), the security of users depends entirely on how hard the hashes are to reverse. MD5 was cracked at over 50 billion hashes per second with an RTX 4090. SHA-256 runs at over 23 billion hashes per second. With a dictionary of 10 billion common passwords, an attacker can crack a complete database of unsalted SHA-256 hashes in under a second. In 2012, LinkedIn leaked 6.5 million SHA-1 password hashes — 90% were cracked within days. A bcrypt hash with cost factor 12 runs at roughly 15,000 hashes per second on the same hardware, making the same attack take centuries. MITRE ATT&CK T1110.002 covers password cracking as an access technique.

## What Gets Flagged

```python
# FLAGGED: MD5 for passwords — ~50 billion hashes/second on modern GPU
import hashlib
hashed = hashlib.md5(password.encode()).hexdigest()

# FLAGGED: SHA-256 for passwords — still billions/second, not suitable for passwords
hashed = hashlib.sha256(password.encode()).hexdigest()
```

```php
<?php
// FLAGGED: md5 for password storage
$hash = md5($password);

// FLAGGED: sha1 for password storage
$hash = sha1($password);
```

```java
// FLAGGED: SHA-1 MessageDigest for passwords
MessageDigest md = MessageDigest.getInstance("SHA-1");
byte[] hash = md.digest(password.getBytes());

// FLAGGED: MD5 MessageDigest for passwords
MessageDigest md2 = MessageDigest.getInstance("MD5");
byte[] hash2 = md2.digest(password.getBytes());
```

```ruby
# FLAGGED: SHA-1 for password storage
require 'digest'
hashed = Digest::SHA1.hexdigest(password)

# FLAGGED: SHA-256 used directly (still too fast for passwords)
hashed = Digest::SHA256.hexdigest(password)
```

```javascript
// FLAGGED: crypto.createHash for passwords — orders of magnitude too fast
const crypto = require('crypto');
const hash = crypto.createHash('sha256').update(password).digest('hex');
```

## Remediation

### Python

```python
# SAFE: Use bcrypt (install: pip install bcrypt)
import bcrypt

def hash_password(password: str) -> bytes:
    return bcrypt.hashpw(password.encode(), bcrypt.gensalt(rounds=12))

def verify_password(password: str, hashed: bytes) -> bool:
    return bcrypt.checkpw(password.encode(), hashed)

# SAFE: Argon2 (preferred for new applications, NIST-recommended)
# install: pip install argon2-cffi
from argon2 import PasswordHasher
ph = PasswordHasher(time_cost=2, memory_cost=65536, parallelism=2)

def hash_password(password: str) -> str:
    return ph.hash(password)

def verify_password(password: str, hashed: str) -> bool:
    return ph.verify(hashed, password)
```

### PHP

```php
<?php
// SAFE: PHP's built-in password_hash uses bcrypt by default
$hash = password_hash($password, PASSWORD_BCRYPT, ['cost' => 12]);

// To verify:
if (password_verify($password, $hash)) {
    // authenticated
}

// SAFE: Argon2 (PHP 7.2+)
$hash = password_hash($password, PASSWORD_ARGON2ID);
```

### Java

```java
// SAFE: BCrypt via Spring Security
import org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder;
BCryptPasswordEncoder encoder = new BCryptPasswordEncoder(12);
String hash = encoder.encode(rawPassword);
boolean matches = encoder.matches(rawPassword, hash);

// SAFE: PBKDF2 with HMAC-SHA256 (built into JDK)
import javax.crypto.spec.PBEKeySpec;
import javax.crypto.SecretKeyFactory;

PBEKeySpec spec = new PBEKeySpec(
    password.toCharArray(),
    salt,          // 16 bytes from SecureRandom
    310_000,       // NIST-recommended minimum iterations for PBKDF2-HMAC-SHA256
    256
);
SecretKeyFactory skf = SecretKeyFactory.getInstance("PBKDF2WithHmacSHA256");
byte[] hash = skf.generateSecret(spec).getEncoded();
```

### Ruby

```ruby
# SAFE: bcrypt-ruby gem
require 'bcrypt'

hashed = BCrypt::Password.create(password, cost: 12)

# To verify:
BCrypt::Password.new(hashed) == password  # true if match
```

### Node.js

```javascript
// SAFE: bcrypt (install: npm install bcrypt)
const bcrypt = require('bcrypt');

async function hashPassword(password) {
    return bcrypt.hash(password, 12);
}

async function verifyPassword(password, hash) {
    return bcrypt.compare(password, hash);
}

// SAFE: Argon2 (install: npm install argon2)
const argon2 = require('argon2');

async function hashPassword(password) {
    return argon2.hash(password, { type: argon2.argon2id });
}
```
