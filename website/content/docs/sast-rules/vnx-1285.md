---
title: "VNX-1285 – Improper Validation of Specified Index, Position, or Offset in Input"
description: "Detect user-supplied array indexes, buffer offsets, or string positions used without bounds validation, leading to out-of-bounds read/write or array index errors."
---

## Overview

Array indexes, buffer offsets, and string positions derived from user input must be validated to be within the valid range `[0, length-1]` before use. Using an unvalidated index can cause out-of-bounds memory access in C/C++ (exploitable as buffer overflow/read), `ArrayIndexOutOfBoundsException` in Java, `IndexError` in Python, or undefined behaviour. **Severity:** High | **CWE:** [CWE-1285 – Improper Validation of Specified Index, Position, or Offset in Input](https://cwe.mitre.org/data/definitions/1285.html)

## Why This Matters

Out-of-bounds array access is one of the most exploited vulnerability classes. In managed languages it typically causes crashes (which can be used for DoS). In C/C++ it can overwrite adjacent memory, enabling arbitrary code execution. Even in safe languages, negative indexes in some frameworks (Python) wrap around the array, allowing access to unintended elements.

## What Gets Flagged

```java
// FLAGGED: No bounds check before array access
int index = Integer.parseInt(request.getParameter("index"));
String item = items[index];  // ArrayIndexOutOfBoundsException if index >= items.length
```

```python
# FLAGGED: User-supplied index without validation
idx = int(request.GET['idx'])
item = my_list[idx]  # can be negative (wraps around) or out of range
```

```c
// FLAGGED: User input as array index in C
int idx = atoi(argv[1]);
char item = data[idx];  // undefined behaviour if idx < 0 or >= sizeof(data)
```

## Remediation

```java
// SAFE: Validate before accessing
int index = Integer.parseInt(request.getParameter("index"));
if (index < 0 || index >= items.length) {
    throw new IllegalArgumentException("Index out of range");
}
String item = items[index];
```

```python
# SAFE: Validate index before use
idx = int(request.GET.get('idx', 0))
if idx < 0 or idx >= len(my_list):
    return HttpResponseBadRequest("Invalid index")
item = my_list[idx]
```

```c
// SAFE: Validate before indexing
int idx = atoi(argv[1]);
if (idx < 0 || (size_t)idx >= ARRAY_SIZE) {
    fprintf(stderr, "Index out of range\n");
    exit(1);
}
char item = data[idx];
```
