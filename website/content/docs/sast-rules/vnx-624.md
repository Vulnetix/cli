## Why This Matters

This weakness class (Executable Regular Expression Error) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-624 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.
