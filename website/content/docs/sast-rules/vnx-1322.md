title: "VNX-1322 – Use of Blocking Code in Single-threaded, Non-blocking Context"
description: "Detect blocking I/O calls (fs.readFileSync, time.sleep, requests.get) used inside asynchronous event-loop contexts (Node.js, Python asyncio) where they starve all other coroutines."
---

## Overview

Event-loop runtimes like Node.js and Python asyncio run on a single thread. When blocking I/O — synchronous file reads, blocking network calls, or `time.sleep()` — is executed in this thread, it prevents all other pending callbacks and coroutines from running until the blocking call returns. Under load this causes request queuing, latency spikes, and effectively denies service to other users. **Severity:** Medium | **CWE:** [CWE-1322 – Use of Blocking Code in Single-threaded, Non-blocking Context](https://cwe.mitre.org/data/definitions/1322.html)

## Why This Matters

Node.js servers handling 1000 requests/second that encounter a 100ms `fs.readFileSync()` call will stall *all* 1000 requests for 100ms. An attacker can trigger the slow path to amplify this into a denial-of-service condition. Python asyncio with a `time.sleep(10)` in a handler blocks the entire event loop for 10 seconds.

## What Gets Flagged

```javascript
// FLAGGED: Synchronous file read in Node.js handler
app.get('/file', (req, res) => {
    const content = fs.readFileSync('/data/' + req.query.name);  // blocks event loop
    res.send(content);
});

// FLAGGED: execSync in request handler
app.post('/run', (req, res) => {
    const result = execSync(req.body.command);  // blocks for duration of command
    res.send(result);
});
```

```python
# FLAGGED: Blocking sleep in async function
async def handler(request):
    time.sleep(1)  # blocks entire event loop for 1 second
    return response

# FLAGGED: Synchronous HTTP request in async context
async def fetch_data():
    response = requests.get('https://api.example.com/data')  # blocks event loop
    return response.json()
```

## Remediation

```javascript
// SAFE: Async file read
app.get('/file', async (req, res) => {
    const content = await fs.promises.readFile('/data/' + req.query.name);
    res.send(content);
});
```

```python
# SAFE: asyncio.sleep for non-blocking pause
async def handler(request):
    await asyncio.sleep(1)  # yields control to event loop
    return response

# SAFE: aiohttp for non-blocking HTTP
import aiohttp

async def fetch_data():
    async with aiohttp.ClientSession() as session:
        async with session.get('https://api.example.com/data') as resp:
            return await resp.json()
```
