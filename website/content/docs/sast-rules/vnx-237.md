## Overview

Constructing XML tags, JSON objects, or protocol messages by concatenating user input without proper encoding allows structural injection. An attacker injects additional tags or key-value pairs that alter the structure of the serialized data. This maps to [CWE-237: Improper Handling of Structural Elements](https://cwe.mitre.org/data/definitions/237.html).

**Severity:** Low | **CWE:** [CWE-237](https://cwe.mitre.org/data/definitions/237.html)

## Remediation

Use proper serialization libraries rather than string concatenation:

```python
# SAFE: use xml.etree or json.dumps
import json
output = json.dumps({"user": user_value, "data": data_value})
```
