title: "VNX-640 – Weak Password Recovery Mechanism for Forgotten Password"
description: "Detect weak password reset token generation using predictable random values or insecure hashing."
---

## Overview

Password reset flows are high-value attack targets because a successful exploit grants full account takeover. Weak reset mechanisms use predictable tokens (based on timestamps or weak random number generators), short numeric codes, or MD5 hashes of usernames — all of which can be guessed or brute-forced within seconds.

**Severity:** High | **CWE:** [CWE-640 – Weak Password Recovery Mechanism for Forgotten Password](https://cwe.mitre.org/data/definitions/640.html)

## Why This Matters

A 4-6 digit numeric reset code (generated with `random.randint`) has at most 1,000,000 possibilities. Without rate limiting, this can be brute-forced in minutes. Token generation using `hashlib.md5(username + timestamp)` is trivially predictable. Exploiting this gives attackers full account takeover on any account they know the email for.

## What Gets Flagged

```python
# FLAGGED: Weak numeric reset code
import random
reset_code = str(random.randint(100000, 999999))

# FLAGGED: MD5 of username as reset token
import hashlib
token = hashlib.md5(username.encode()).hexdigest()

# FLAGGED: Math.random() for security token (not cryptographically secure)
reset_token = Math.random().toString(36).substring(2)
```

```php
<?php
// FLAGGED: Predictable token from user-controlled data
$reset_token = md5($username . time());
```

## Remediation

Use cryptographically secure token generation with sufficient entropy:

```python
# SAFE: cryptographically secure 32-byte token
import secrets
reset_token = secrets.token_urlsafe(32)

# Store with expiry (e.g., 1 hour)
user.reset_token = reset_token
user.reset_token_expires = datetime.utcnow() + timedelta(hours=1)
user.save()

# On verification: constant-time comparison
if secrets.compare_digest(provided_token, user.reset_token):
    if datetime.utcnow() < user.reset_token_expires:
        allow_password_reset()
```

```javascript
// SAFE: Node.js crypto random bytes
const crypto = require('crypto');
const resetToken = crypto.randomBytes(32).toString('hex');
```
