## Overview

Detects deep inheritance chains in Java that extend beyond common framework base classes. Deep inheritance makes method resolution unpredictable, security controls in parent classes easy to accidentally override, and code harder to audit.

**Severity:** Low | **CWE:** [CWE-1074](https://cwe.mitre.org/data/definitions/1074.html)

## Why This Matters

Review the flagged patterns and assess whether they represent security risks in your codebase. Even quality/maintainability weaknesses can have security implications when they affect how security-critical code is written and audited.

2. Refactor code to follow the recommended patterns described in the rule metadata.
