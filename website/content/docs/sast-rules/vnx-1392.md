---
title: "VNX-1392 – Use of Default Credentials"
description: "Detect authentication logic that compares credentials against hardcoded default values like 'admin'/'admin' or 'root'/'root', which remain accessible to any attacker who knows the defaults."
---

## Overview

When authentication logic compares user-supplied credentials against hardcoded literal values, those literal values become permanent backdoors. Unlike CWE-1391 (credentials in configuration), CWE-1392 focuses on the comparison logic itself: `if username == 'admin' and password == 'admin'`. This is unremovable without a code change and invisible to secrets scanners that only look at variable assignments. **Severity:** Critical | **CWE:** [CWE-1392 – Use of Default Credentials](https://cwe.mitre.org/data/definitions/1392.html)

## Why This Matters

Authentication bypass via hardcoded credential checks is consistently found in network devices, embedded systems, and legacy web applications. CVE-2021-20090 (Arcadyan router firmware) used hardcoded credentials. The CISA Known Exploited Vulnerabilities catalog regularly includes hardcoded credential CVEs. These checks survive refactors and are often introduced via "temporary" debug backdoors.

## What Gets Flagged

```python
# FLAGGED: Hardcoded credential comparison
def login(username, password):
    if username == 'admin' and password == 'admin':
        return True  # hardcoded backdoor
    return check_database(username, password)
```

```javascript
// FLAGGED: Default credential check in auth middleware
if (req.body.username === 'admin' && req.body.password === 'password') {
    req.session.authenticated = true;
}
```

```java
// FLAGGED: Literal credential comparison
if (username.equals("admin") && password.equals("admin")) {
    return AuthResult.SUCCESS;
}
```

## Remediation

All authentication must go through a credential store that uses password hashing:

```python
# SAFE: Hash-based credential verification
from argon2 import PasswordHasher
from myapp.models import User

ph = PasswordHasher()

def login(username, password):
    try:
        user = User.objects.get(username=username)
        ph.verify(user.password_hash, password)
        return user
    except (User.DoesNotExist, argon2.exceptions.VerifyMismatchError):
        return None  # constant-time failure
```

```javascript
// SAFE: bcrypt comparison
const bcrypt = require('bcrypt');

async function login(username, password) {
    const user = await User.findOne({ username });
    if (!user) return null;
    const match = await bcrypt.compare(password, user.passwordHash);
    return match ? user : null;
}
```
