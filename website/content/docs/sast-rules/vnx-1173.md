## Overview

VNX-1173 maps to [CWE-1173: Improper Use of Validation Framework](https://cwe.mitre.org/data/definitions/1173.html). Detects source patterns associated with CWE-1173 (Improper Use of Validation Framework). Each finding should be manually reviewed for exploitability in context.

**Severity:** Low | **CWE:** [CWE-1173](https://cwe.mitre.org/data/definitions/1173.html) | **Languages:** java

## Why This Matters

This weakness class (Improper Use of Validation Framework) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-1173 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```java
// FLAGGED: contains '@Valid' pattern
@Valid
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-1173 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-1173` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
