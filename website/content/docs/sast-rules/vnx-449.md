## Overview

VNX-449 maps to [CWE-449: The UI Performs the Wrong Action](https://cwe.mitre.org/data/definitions/449.html). Detects source patterns associated with CWE-449 (The UI Performs the Wrong Action). Each finding should be manually reviewed for exploitability in context.

**Severity:** Low | **CWE:** [CWE-449](https://cwe.mitre.org/data/definitions/449.html) | **Languages:** c, cpp

## Why This Matters

This weakness class (The UI Performs the Wrong Action) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-449 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```c
// FLAGGED: contains 'str ' pattern
str 
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-449 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-449` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
