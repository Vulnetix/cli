---
title: "VNX-694 – Duplicate Identifier"
description: "Detect patterns where session IDs or tokens are set equal to user-controlled identifiers, creating predictable session tokens."
---

## Overview

Session identifiers must be cryptographically random and independent of any user-controlled or predictable values. When a session ID is set to a user ID, username, or other guessable value — intentionally or by copy-paste error — any user can impersonate any other user by setting their session cookie to the target's known identifier.

**Severity:** Low | **CWE:** [CWE-694 – Duplicate Identifier](https://cwe.mitre.org/data/definitions/694.html)

## Why This Matters

Setting `session_id = user_id` means that knowing a target user's user ID (often a sequential integer or email address) is sufficient to hijack their session. An attacker who discovers user IDs through IDOR or enumeration can immediately compromise every discovered account with a trivial HTTP request.

## What Gets Flagged

```python
# FLAGGED: Session ID set to user ID — predictable and enumerable
session_id = user_id
request.session['session_id'] = user_id

# FLAGGED: Token set to username
session_id = username
token = user_id
```

```javascript
// FLAGGED: Session token equals user ID
const sessionId = userId;
req.session.token = userId;
```

## Remediation

Generate session identifiers independently using cryptographically secure random values:

```python
# SAFE: Cryptographically random session ID
import secrets
session_id = secrets.token_urlsafe(32)
request.session['session_id'] = session_id

# Store the association server-side
SessionStore.set(session_id, {'user_id': user_id, 'expires': ...})
```

```javascript
// SAFE: crypto random bytes for session token
const crypto = require('crypto');
const sessionId = crypto.randomBytes(32).toString('hex');
req.session.id = sessionId;
// Store user_id in server-side session store, keyed by sessionId
```

The session ID should be the key to look up the user ID in a server-side store — never the user ID itself.
