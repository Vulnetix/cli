## Overview

VNX-262 maps to [CWE-262: Not Using Password Aging](https://cwe.mitre.org/data/definitions/262.html). Detects source patterns associated with CWE-262 (Not Using Password Aging). Each finding should be manually reviewed for exploitability in context.

**Severity:** Medium | **CWE:** [CWE-262](https://cwe.mitre.org/data/definitions/262.html) | **Languages:** java, node, python

## Why This Matters

This weakness class (Not Using Password Aging) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-262 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```python
// FLAGGED: contains 'password_expires = False' pattern
password_expires = False
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-262 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-262` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
