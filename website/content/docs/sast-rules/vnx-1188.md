## Overview

VNX-1188 maps to [CWE-1188: Initialization of a Resource with an Insecure Default](https://cwe.mitre.org/data/definitions/1188.html). Detects source patterns associated with CWE-1188 (Initialization of a Resource with an Insecure Default). Each finding should be manually reviewed for exploitability in context.

**Severity:** Medium | **CWE:** [CWE-1188](https://cwe.mitre.org/data/definitions/1188.html) | **Languages:** node, python

## Why This Matters

This weakness class (Initialization of a Resource with an Insecure Default) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-1188 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```python
// FLAGGED: contains 'DEBUG = True' pattern
DEBUG = True
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-1188 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-1188` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
