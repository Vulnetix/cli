## Overview

Regular expression denial of service (ReDoS) exploits regex patterns with exponential worst-case matching time. Patterns like `(a+)+`, `(a|aa)+`, or `([a-z]+)*` can take milliseconds on short inputs but minutes or hours on crafted 50-character inputs. When applied to user-supplied data in a web request handler, a single request can freeze the process. **Severity:** High | **CWE:** [CWE-1333 – Inefficient Regular Expression Complexity (ReDoS)](https://cwe.mitre.org/data/definitions/1333.html)

## Why This Matters

ReDoS has been exploited in production systems: Stack Overflow was taken down by a ReDoS in 2016 (`/^[\s\u200c]+|[\s\u200c]+$/` applied to a long string). The npm `ua-parser-js` library had multiple ReDoS CVEs. OWASP lists ReDoS as a significant denial-of-service vector. The attack requires only a single malformed HTTP request.

## What Gets Flagged

```python
# FLAGGED: Catastrophic backtracking pattern
import re
pattern = re.compile(r'(a+)+$')
pattern.match(user_input)  # 'aaaaaaaaaaaaaaaaaaaaaaaaaaaa!' takes exponential time

# FLAGGED: Regex applied to user input
result = re.search(r'([a-zA-Z]+)*', user_data)
```

```javascript
// FLAGGED: Catastrophic backtracking
const dangerous = /(a+)+/.test(userInput);
const also_bad = /^([\s\S]*)*$/.test(req.body.data);
```

```java
// FLAGGED: Nested quantifiers
Pattern p = Pattern.compile("(\\w+\\s*)*=");
p.matcher(userInput).matches();
```

## Remediation

1. **Audit regex patterns** with tools like [safe-regex](https://github.com/substack/safe-regex), [vuln-regex-detector](https://github.com/nicowillis/vuln-regex-detector), or [rxxr2](https://github.com/superhuman/rxxr2).

2. **Rewrite catastrophic patterns** to be possessive or atomic:
```python
# UNSAFE: (a+)+
# SAFE equivalent using atomic-like approach (avoid nested repetition)
re.compile(r'a+')  # match sequence without nested groups

# Or use re2 which guarantees linear time
import re2
pattern = re2.compile(r'(a+)+')  # re2 rewrites to linear
```

3. **Set regex timeouts** where the runtime supports it:
```javascript
// Use a timeout wrapper
const { execFileSync } = require('child_process');
// Or use re2 npm package
const RE2 = require('re2');
const safe = new RE2('(a+)+');
```

4. **Validate input length before applying complex patterns:**
```python
if len(user_input) > 1000:
    raise ValueError("Input too long")
result = re.match(pattern, user_input)
```
