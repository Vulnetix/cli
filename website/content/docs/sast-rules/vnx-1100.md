## Overview

Detects direct usage of platform-specific system calls, OS APIs, and environment detection scattered through business logic. System-dependent functions should be isolated in platform abstraction layers to prevent security bypass when running on unexpected platforms.

**Severity:** Low | **CWE:** [CWE-1100](https://cwe.mitre.org/data/definitions/1100.html)

## Why This Matters

Review the flagged patterns in context. Even quality and maintainability weaknesses can have security implications when they affect how security-critical code is structured, audited, and tested.

## Remediation

1. Review all flagged instances in context.
2. Refactor code to follow the recommended patterns described in the rule.
3. Consult [CWE-1100](https://cwe.mitre.org/data/definitions/1100.html) for authoritative guidance.
