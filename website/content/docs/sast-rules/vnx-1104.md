---

## Overview

Detects imports of known abandoned, deprecated, or end-of-life third-party packages. Unmaintained dependencies do not receive security patches, making known vulnerabilities permanently exploitable in codebases that use them.

**Severity:** Medium | **CWE:** [CWE-1104 – Use of Unmaintained Third-Party Components](https://cwe.mitre.org/data/definitions/1104.html)

## Why This Matters

The `request` npm package (deprecated 2020) had multiple security advisories and is no longer patched. The Python `md5` module was removed in Python 3. Using these packages means running code with known, unpatched vulnerabilities.

## What Gets Flagged

```javascript
// FLAGGED: deprecated npm package
const request = require('request');
```

```python
# FLAGGED: removed Python 2 module
import md5
import urllib2
```

```go
// FLAGGED: deprecated Go package
import "io/ioutil"
import "golang.org/x/net/context"
```

## Remediation

| Deprecated | Replacement |
|---|---|
| `request` (npm) | `node-fetch`, `axios`, or `undici` |
| Python `md5` | `hashlib.md5()` |
| Python `urllib2` | `urllib.request` |
| Go `io/ioutil` | `io` and `os` packages directly |
| Go `golang.org/x/net/context` | `context` standard library |
