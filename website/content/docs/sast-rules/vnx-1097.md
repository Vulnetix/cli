## Overview

Detects Java entity/model classes annotated with @Entity or implementing Comparable that do not override equals() and hashCode(). Without proper equals/hashCode, identity checks on persistent objects can silently use object reference equality, causing security logic based on object comparison to behave incorrectly.

**Severity:** Low | **CWE:** [CWE-1097](https://cwe.mitre.org/data/definitions/1097.html)

## Why This Matters

Review the flagged patterns in context. Even quality and maintainability weaknesses can have security implications when they affect how security-critical code is structured, audited, and tested.
