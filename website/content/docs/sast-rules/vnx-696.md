description: "Detect Time-of-Check to Time-of-Use (TOCTOU) race conditions where os.access() is used to check permissions before open()."
---

## Overview

Time-of-Check to Time-of-Use (TOCTOU) is a class of race condition where a security check (checking if a file exists, checking permissions, checking file type) and the subsequent use of the resource are not atomic. An attacker who can manipulate the filesystem between the check and the use can substitute a malicious resource for the legitimate one that was checked.

**Severity:** Low | **CWE:** [CWE-696 – Incorrect Behavior Order](https://cwe.mitre.org/data/definitions/696.html)

## Why This Matters

The classic Python TOCTOU: `os.access('/tmp/file', os.R_OK)` followed by `open('/tmp/file')` — between the `access()` call and the `open()` call, an attacker can replace `/tmp/file` with a symlink pointing to `/etc/shadow`. The `access()` check passes because the original file was readable; the `open()` opens the sensitive target file. Privilege escalation attacks and symlink attacks in `/tmp` exploit exactly this pattern.

## What Gets Flagged

```python
# FLAGGED: TOCTOU — check then open is not atomic
import os
if os.access(user_path, os.R_OK):   # check
    with open(user_path) as f:       # use — race window here
        return f.read()

# FLAGGED: Similar pattern with os.path.exists
if os.path.exists(filename):
    with open(filename) as f:
        data = f.read()
```

```c
// FLAGGED: access() before open() — classic TOCTOU
if (access(path, R_OK) == 0) {
    fd = open(path, O_RDONLY);  // race condition between these calls
}
```

## Remediation

Open the file directly and handle permission errors as exceptions, eliminating the check-use window:

```python
# SAFE: Open directly, handle errors — no TOCTOU window
try:
    with open(user_path) as f:
        return f.read()
except PermissionError:
    return HttpResponseForbidden()
except FileNotFoundError:
    return HttpResponseNotFound()
```

```c
// SAFE: Open directly, check return value
fd = open(path, O_RDONLY);
if (fd < 0) {
    perror("open");
    return -1;
}
```

For directory operations, use `openat()` with file descriptor anchoring to prevent symlink attacks.
