---
title: "VNX-759 – Use of a One-Way Hash without a Salt"
description: "Detect unsalted password hashing using SHA-256, SHA-512, MD5, or similar fast hash algorithms that are vulnerable to precomputed rainbow table attacks."
---

## Overview

This rule flags password hashing that uses fast cryptographic hash functions (SHA-256, SHA-512, MD5, SHA-1) without a unique per-user salt. Without a salt, identical passwords produce identical hashes — allowing attackers to use precomputed rainbow tables to crack multiple passwords at once after a database breach. Even with a salt, fast hash functions are millions of times faster than purpose-built password hashing algorithms, making brute force feasible. This maps to [CWE-759: Use of a One-Way Hash without a Salt](https://cwe.mitre.org/data/definitions/759.html).

**Severity:** High | **CWE:** [CWE-759](https://cwe.mitre.org/data/definitions/759.html)

## Why This Matters

When a database is breached, unsalted SHA-256 password hashes can be cracked for the majority of users in hours using commodity GPU hardware and publicly available rainbow tables. The SHA-256 of "password123" is the same for every user who chose that password — crack it once, crack them all. Modern password hashing algorithms (bcrypt, Argon2, scrypt) are specifically designed to be slow and are the only appropriate choice for password storage.

## What Gets Flagged

```python
# FLAGGED: unsalted SHA-256 of password
import hashlib
stored = hashlib.sha256(password.encode()).hexdigest()

# FLAGGED: MD5 of password
stored = hashlib.md5(password.encode()).hexdigest()
```

```java
// FLAGGED: MessageDigest without salt
MessageDigest md = MessageDigest.getInstance("SHA-256");
byte[] hash = md.digest(password.getBytes());
```

```ruby
# FLAGGED: unsalted Digest
require 'digest'
stored = Digest::SHA256.hexdigest(password)
```

```javascript
// FLAGGED: crypto.createHash without salt
const hash = crypto.createHash('sha256').update(password).digest('hex');
```

## Remediation

Use purpose-built password hashing algorithms:

```python
# SAFE: bcrypt via passlib
from passlib.hash import bcrypt
stored_hash = bcrypt.hash(password)
is_valid = bcrypt.verify(password, stored_hash)

# SAFE: Argon2 (recommended by OWASP)
from argon2 import PasswordHasher
ph = PasswordHasher()
stored_hash = ph.hash(password)
ph.verify(stored_hash, password)
```

```java
// SAFE: Spring Security BCryptPasswordEncoder
PasswordEncoder encoder = new BCryptPasswordEncoder(12);
String stored = encoder.encode(password);
boolean valid = encoder.matches(submitted, stored);
```

```javascript
// SAFE: bcrypt
const bcrypt = require('bcrypt');
const stored = await bcrypt.hash(password, 12);
const valid = await bcrypt.compare(submitted, stored);
```
