## Overview

VNX-654 maps to [CWE-654: Reliance on a Single Factor in a Security Decision](https://cwe.mitre.org/data/definitions/654.html). Detects source patterns associated with CWE-654 (Reliance on a Single Factor in a Security Decision). Each finding should be manually reviewed for exploitability in context.

**Severity:** Low | **CWE:** [CWE-654](https://cwe.mitre.org/data/definitions/654.html) | **Languages:** node, python

## Why This Matters

This weakness class (Reliance on a Single Factor in a Security Decision) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-654 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```python
// FLAGGED: contains 'token_only' pattern
token_only
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-654 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-654` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
