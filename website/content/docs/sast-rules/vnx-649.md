---
title: "VNX-649 – Reliance on Obfuscation or Encryption of Security-Relevant Inputs without Integrity Checking"
description: "Detect use of base64 encoding or simple encryption to 'protect' sensitive values without proper authentication."
---

## Overview

Base64 encoding passwords, secrets, or security tokens before storing or transmitting them provides no security — base64 is trivially reversible. Similarly, encrypting values without an integrity check (MAC) makes them vulnerable to padding oracle attacks or bit-flipping attacks. Treating obfuscation as equivalent to encryption, or encryption as equivalent to authentication, is a fundamental security design flaw.

**Severity:** Medium | **CWE:** [CWE-649 – Reliance on Obfuscation or Encryption of Security-Relevant Inputs without Integrity Checking](https://cwe.mitre.org/data/definitions/649.html)

## Why This Matters

Any attacker who intercepts a base64-encoded password can trivially decode it with a single command. "Encrypted" cookies without an HMAC can be decrypted or modified by attackers who perform padding oracle attacks (classic ASP.NET VIEWSTATE) or bit-flipping attacks. This pattern gives developers false confidence that data is "protected."

## What Gets Flagged

```python
# FLAGGED: base64 encoding a password (trivially reversible)
import base64
stored_password = base64.b64encode(secret.encode())

# FLAGGED: base64 "securing" a session token
encoded_token = base64.b64encode(password.encode()).decode()
```

```javascript
// FLAGGED: btoa to "protect" password
const encodedPassword = btoa(password);
localStorage.setItem('auth', encodedPassword);
```

```php
<?php
// FLAGGED: base64_encode on password
$stored = base64_encode($password);
```

## Remediation

Use proper cryptographic techniques appropriate to the security goal:

```python
# SAFE: Hash passwords with bcrypt/argon2
import bcrypt
hashed = bcrypt.hashpw(password.encode(), bcrypt.gensalt(rounds=12))

# SAFE: Authenticated encryption (AES-GCM provides confidentiality + integrity)
from cryptography.hazmat.primitives.ciphers.aead import AESGCM
import os
key = AESGCM.generate_key(bit_length=256)
aesgcm = AESGCM(key)
nonce = os.urandom(12)
ciphertext = aesgcm.encrypt(nonce, plaintext, associated_data)
# AESGCM.decrypt() will raise if the ciphertext has been tampered with
```

For transmitting sensitive values: use TLS (which provides both confidentiality and integrity), not base64.
