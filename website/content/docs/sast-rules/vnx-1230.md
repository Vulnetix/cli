## Overview

VNX-1230 maps to [CWE-1230: Exposure of Sensitive Information Through Metadata](https://cwe.mitre.org/data/definitions/1230.html). Detects source patterns associated with CWE-1230 (Exposure of Sensitive Information Through Metadata). Each finding should be manually reviewed for exploitability in context.

**Severity:** Medium | **CWE:** [CWE-1230](https://cwe.mitre.org/data/definitions/1230.html) | **Languages:** node, python

## Why This Matters

This weakness class (Exposure of Sensitive Information Through Metadata) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-1230 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```python
// FLAGGED: contains 'exifread' pattern
exifread
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-1230 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-1230` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
