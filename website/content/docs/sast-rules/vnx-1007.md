## Overview

VNX-1007 maps to [CWE-1007: Insufficient Visual Distinction of Homoglyphs Presented to User](https://cwe.mitre.org/data/definitions/1007.html). **Severity:** Low | **CWE:** [CWE-1007](https://cwe.mitre.org/data/definitions/1007.html) | **Languages:** java, node, python

## Why This Matters

This weakness class (Insufficient Visual Distinction of Homoglyphs Presented to User) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-1007 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```python
// FLAGGED: contains 'Аdmin' pattern
Аdmin
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-1007 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-1007` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
