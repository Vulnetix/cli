---
title: "VNX-256 – Plaintext Storage of a Password"
description: "Detect passwords stored in plaintext in database operations or model assignments without hashing."
---

## Overview

Storing passwords in plaintext means a single database breach exposes all user credentials. There is no legitimate reason to store a recoverable password — authentication only requires comparing a submitted password to a stored hash. This maps to [CWE-256: Plaintext Storage of a Password](https://cwe.mitre.org/data/definitions/256.html).

**Severity:** High | **CWE:** [CWE-256](https://cwe.mitre.org/data/definitions/256.html)

## Remediation

```python
# SAFE: bcrypt hash before storing
import bcrypt
hashed = bcrypt.hashpw(password.encode(), bcrypt.gensalt(rounds=12))
user.password = hashed.decode()
user.save()
```

```java
// SAFE: BCrypt in Spring Security
BCryptPasswordEncoder encoder = new BCryptPasswordEncoder(12);
user.setPassword(encoder.encode(rawPassword));
```
