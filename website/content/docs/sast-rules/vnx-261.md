title: "VNX-261 – Weak Encoding for Password"
description: "Detect Base64 or ROT13 encoding used as password 'protection'."
---

## Overview

Base64 and ROT13 are trivially reversible encodings that provide no real protection for passwords. Any attacker who obtains the stored value can decode it in milliseconds without any key material. This maps to [CWE-261: Weak Encoding for Password](https://cwe.mitre.org/data/definitions/261.html).

**Severity:** High | **CWE:** [CWE-261](https://cwe.mitre.org/data/definitions/261.html)

## Remediation

```python
# SAFE: proper password hashing
from passlib.hash import argon2
stored = argon2.hash(password)
# Verify:
argon2.verify(password, stored)
```

```php
<?php
// SAFE: PHP password_hash
$hash = password_hash($password, PASSWORD_ARGON2ID);
// Verify:
password_verify($input, $hash);
```
