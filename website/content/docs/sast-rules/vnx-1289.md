title: "VNX-1289 – Improper Validation of Unsafe Equivalence in Input"
description: "Detect URL decoding and path normalisation operations that may be applied after security checks, allowing encoded path traversal sequences (%2e%2e%2f) and other equivalent representations to bypass filters."
---

## Overview

Security filters applied to encoded or non-canonical input can be bypassed by using equivalent representations. A filter that blocks `../` is trivially bypassed with `%2e%2e%2f` (URL-encoded), `..%2f` (mixed), or `%2e%2e/` (partial encoding). The canonical fix is to fully decode and canonicalise (normalise) input *before* applying security checks. **Severity:** Medium | **CWE:** [CWE-1289 – Improper Validation of Unsafe Equivalence in Input](https://cwe.mitre.org/data/definitions/1289.html)

## Why This Matters

Path traversal via encoded characters has been exploited in numerous CVEs including the Apache HTTP Server path traversal (CVE-2021-41773) and IIS Unicode path traversal (MS01-026). The order of operations — decode then validate, not validate then decode — is critical and easy to get wrong.

## What Gets Flagged

```python
# FLAGGED: URL decoding after check (wrong order)
path = request.GET['file']
if '../' in path:
    return HttpResponseForbidden()
# Attacker uses %2e%2e%2f which passes the check, then:
decoded_path = urllib.parse.unquote(path)  # too late — check already passed
```

```javascript
// FLAGGED: decodeURIComponent without prior validation
const userPath = req.query.path;
const decoded = decodeURIComponent(userPath);
// If ../blocked check was before this decode, it can be bypassed
```

## Remediation

```python
# SAFE: Decode first, then validate, then canonicalise
import os
import urllib.parse

def safe_file_path(user_input, base_dir):
    # Step 1: decode
    decoded = urllib.parse.unquote(user_input)
    # Step 2: canonicalise (resolve . and ..)
    absolute = os.path.realpath(os.path.join(base_dir, decoded))
    # Step 3: check it is within the allowed base directory
    if not absolute.startswith(os.path.realpath(base_dir) + os.sep):
        raise PermissionError("Path traversal detected")
    return absolute
```

```go
// SAFE: Use filepath.Clean then prefix check
import (
    "path/filepath"
    "strings"
)

func safeJoin(base, userInput string) (string, error) {
    clean := filepath.Clean(filepath.Join(base, userInput))
    if !strings.HasPrefix(clean, filepath.Clean(base)+string(filepath.Separator)) {
        return "", fmt.Errorf("path traversal detected")
    }
    return clean, nil
}
```
