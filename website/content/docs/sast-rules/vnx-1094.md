## Overview

Detects SQL query patterns that are likely to cause full table scans or index range scans across very wide ranges, such as date ranges without upper bounds, wildcard prefix searches, and unbounded inequality comparisons.

**Severity:** Low | **CWE:** [CWE-1094](https://cwe.mitre.org/data/definitions/1094.html)

## Why This Matters

Review the flagged patterns in context. Even quality and maintainability weaknesses can have security implications when they affect how security-critical code is structured, audited, and tested.

## Remediation

1. Review all flagged instances in context.
2. Refactor code to follow the recommended patterns described in the rule.
3. Consult [CWE-1094](https://cwe.mitre.org/data/definitions/1094.html) for authoritative guidance.
