## Overview

VNX-1341 maps to [CWE-1341: Multiple Releases of Same Resource or Handle](https://cwe.mitre.org/data/definitions/1341.html). Detects source patterns associated with CWE-1341 (Multiple Releases of Same Resource or Handle). Each finding should be manually reviewed for exploitability in context.

**Severity:** Medium | **CWE:** [CWE-1341](https://cwe.mitre.org/data/definitions/1341.html) | **Languages:** c

## Why This Matters

This weakness class (Multiple Releases of Same Resource or Handle) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-1341 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```c
// FLAGGED: contains 'goto' pattern
goto
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-1341 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-1341` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
