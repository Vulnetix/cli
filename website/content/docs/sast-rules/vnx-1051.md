## Overview

Detects hardcoded IP addresses, hostnames, and network endpoints embedded directly in source code. Hardcoded network configuration prevents environment-specific deployment, exposes internal network topology, and cannot be changed without modifying and redeploying code.

**Severity:** Medium | **CWE:** [CWE-1051 – Initialization with Hard-Coded Network Resource Configuration Data](https://cwe.mitre.org/data/definitions/1051.html)

## Why This Matters

Internal IP addresses in source code expose network topology to attackers who can access the code. Hardcoded `localhost` prevents the code from running correctly in Docker or Kubernetes environments. These values need to change between development, staging, and production but cannot do so without code changes.

## What Gets Flagged

```python
# FLAGGED: hardcoded internal IP
db_host = "192.168.1.100"
redis_host = "10.0.0.5"
api_url = "http://localhost:8080"
```

## Remediation

```python
# SAFE: use environment variables
import os
db_host = os.environ.get("DB_HOST", "localhost")
api_url = os.environ.get("API_URL", "http://localhost:8080")
```
