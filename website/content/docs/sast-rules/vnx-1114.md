## Overview

Detects mixed indentation (tabs mixed with spaces) in security-sensitive Python code. Python's indentation-significant syntax means mixed whitespace can cause code to execute in unexpected ways, potentially skipping security checks.

**Severity:** Low | **CWE:** [CWE-1114](https://cwe.mitre.org/data/definitions/1114.html)

## Why This Matters

Review the flagged patterns in context. Even quality and maintainability weaknesses can have security implications when they affect how security-critical code is structured, audited, and tested.

## Remediation

1. Review all flagged instances in context.
2. Refactor code to follow the recommended patterns described in the rule.
3. Consult [CWE-1114](https://cwe.mitre.org/data/definitions/1114.html) for authoritative guidance.
