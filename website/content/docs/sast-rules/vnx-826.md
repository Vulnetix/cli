## Overview

VNX-826 maps to [CWE-826: Premature Release of Resource During Expected Lifetime](https://cwe.mitre.org/data/definitions/826.html). Detects source patterns associated with CWE-826 (Premature Release of Resource During Expected Lifetime). Each finding should be manually reviewed for exploitability in context.

**Severity:** Low | **CWE:** [CWE-826](https://cwe.mitre.org/data/definitions/826.html) | **Languages:** c, cpp

## Why This Matters

This weakness class (Premature Release of Resource During Expected Lifetime) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-826 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```c
// FLAGGED: contains 'free(' pattern
free(
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-826 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-826` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
