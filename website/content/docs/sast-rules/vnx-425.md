## Overview

VNX-425 maps to [CWE-425: Direct Request ('Forced Browsing')](https://cwe.mitre.org/data/definitions/425.html). Detects source patterns associated with CWE-425 (Direct Request ('Forced Browsing')). Each finding should be manually reviewed for exploitability in context.

**Severity:** Medium | **CWE:** [CWE-425](https://cwe.mitre.org/data/definitions/425.html) | **Languages:** python

## Why This Matters

This weakness class (Direct Request ('Forced Browsing')) creates a concrete exploit surface: the rule searches for the concrete source-level patterns most commonly associated with CWE-425 and surfaces them for review. Each finding should be evaluated in context — the rule catches the pattern, not the context.

## What Gets Flagged

```python
// FLAGGED: contains '/admin/' pattern
/admin/
```

## Remediation

1. Review each flagged line and determine whether the pattern represents a real instance of CWE-425 or a false positive.
2. Replace the flagged construct with a documented safe alternative appropriate to your language and framework.
3. For confirmed false positives, add a `# vulnetix-ignore: VNX-425` comment on the line.
4. Ensure equivalent test coverage exists to prevent regression.
