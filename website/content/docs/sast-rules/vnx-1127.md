## Overview

Detects inline vulnerability annotations that use incorrect or imprecise CWE/CVE categorisations. Miscategorised vulnerabilities may be assigned incorrect remediation priority or incorrect fix guidance, leaving the actual vulnerability unaddressed.

**Severity:** Info | **CWE:** [CWE-1127](https://cwe.mitre.org/data/definitions/1127.html)

## Why This Matters

Review the flagged patterns in context. Even quality and maintainability weaknesses can have security implications when they affect how security-critical code is structured, audited, and tested.

## Remediation

1. Review all flagged instances in context.
2. Refactor code to follow the recommended patterns described in the rule.
3. Consult [CWE-1127](https://cwe.mitre.org/data/definitions/1127.html) for authoritative guidance.
