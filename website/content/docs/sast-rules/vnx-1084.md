## Overview

Detects functions that perform multiple file open/read/write operations in a single function body. Multiple file operations in one function make it difficult to enforce proper access controls, validate paths, and handle errors consistently.

**Severity:** Low | **CWE:** [CWE-1084](https://cwe.mitre.org/data/definitions/1084.html)

## Why This Matters

Review the flagged patterns in context. Even quality and maintainability weaknesses can have security implications when they affect how security-critical code is structured, audited, and tested.

## Remediation

1. Review all flagged instances in context.
2. Refactor code to follow the recommended patterns described in the rule.
3. Consult [CWE-1084](https://cwe.mitre.org/data/definitions/1084.html) for authoritative guidance.
